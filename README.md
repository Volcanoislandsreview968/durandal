# Durandal

Durandal is a modern system monitor for Linux written in Go. It is designed to be fast, efficient, and easy to use. 

Durandal is designed to be pretty and minimal. It is written in Go and uses Bubbletea for the UI. I treid to make a system monitor that is inspired by the design of the new MArathon games style, which is why it is called Durandal. I wouldn't say I achieved that, but I had fun trying.

As I am trying to get into learning Go. Durandal is, much like synx, also a learning project for me so I have made use of llms in its creation.
